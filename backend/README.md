# Backend (Express)

Razorpay API backend for order creation and payment verification.

## Run

```bash
npm install
npm run dev
```

Server runs on `http://localhost:8787`.

## Required env (`backend/.env`)

```bash
RAZORPAY_KEY_ID=...
RAZORPAY_KEY_SECRET=...
PUBLIC_RAZORPAY_KEY_ID=...
PORT=8787
```
